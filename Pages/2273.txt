F-1 Student Forms
 



Form
Description



This form is used to request new I-20s for dependents who wish to enter the U.S. on an F-2 visa.



This form is to be used when an F-1 student’s academic information such as their degree level has changed, so a new I-20 can be issued on behalf of the student.



This form is to be used when an F-1 student’s academic major has changed, so a new I-20 can be issued on behalf of the student.



This form is to be used when a change of non-immigrant status to F-1 status is requested.




This form is to be used when a continuing F-1 student is requesting Curricular Practical Training.




This form is to be used when a continuing F-1 student is required to apply for a social security number.



This form is to be used when an F-1 student requires additional time to complete their academic program, due to an academic reason.



This form is to be used by an F-1 student who wishes to transfer out of UIC to another U.S. institution.



 
J-1 Student Forms
 



Form
Description



This form is to be used to by a J-1 student to request off-campus work authorization in the form of Academic Training.



This form is used to request new DS-2019s for dependents who wish to enter the U.S. on a J-2 visa.



This form is to be used when a J-1 student’s academic information such as their degree level has changed, so a new DS-2019 can be issued on behalf of the student.



This form is to be used when a J-1 student’s academic major has changed, so a new DS-2019 can be issued on behalf of the student.



This form is to be used when a change of non-immigrant status to J-1 status is requested.



This form is to be used when a J-1 student is applying for employment authorization with USCIS due to severe economic hardship.




This form is to be used when a J-1 student is required to apply for a social security number.



This form is to be used when a J-1 student wishes to obtain on-campus work authorization.



This form is to be used when a J-1 student requires additional time to complete their academic program, due to an academic reason.



This form is to be used by a J-1 student who wishes to transfer out of UIC to another U.S. institution.



 
General Forms
 



Form
Description



This form is to be used by an F-1 or a J-1 student to notify OIS that the student is either leaving the U.S. to return to their home country, or changing to a new status within the U.S.



This form is to be used when an F-1 or J-1 student needs to replace a lost or stolen I-20 or DS-2019, request a status verification or social security letter, or to request a travel signature.



This form is to be used when an F-1 or J-1 student needs to certify sufficient funding for the issuance of an updated I-20 or DS-2019.



This form is to be used only by an F-1 student to inform OIS of changes to employment and personal information while on OPT (Optional Practical Training).



This form is to be used only by an F-1 student to inform OIS of changes to employment and personal information while on STEM Extension OPT (STEM Extension Optional Practical Training).
